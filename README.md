## 开发/调试方法技巧

```bash
#克隆本项目 
git clone  https://github.com/BleethNie/appstore.git

# 添加代码，新增应用

# 推送应用到1panel服务器所在目录
rsync -avzP  -e 'ssh -p 52222'  apps/* root@192.168.1.90:/home/1panel/resource/apps/local/

```
